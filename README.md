🎓 Projeto de Banco de Dados – Sistema de Gestão Escolar
Durante meus estudos em Banco de Dados, desenvolvi um projeto de modelagem e estruturação de um banco de dados para um sistema de gestão escolar.
O objetivo foi praticar conceitos fundamentais de SQL e organização de dados em sistemas reais, simulando a estrutura administrativa e acadêmica de uma escola.
📊 O banco de dados foi estruturado com tabelas relacionadas para gerenciar informações como:
• Setores da escola
• Cargos e funcionários
• Salas de aula
• Turmas
• Cadastro de alunos
Durante o desenvolvimento trabalhei com conceitos importantes como:
✔ Criação de banco de dados
✔ Estruturação de tabelas
✔ Definição de chaves primárias (Primary Key)
✔ Relacionamentos com chaves estrangeiras (Foreign Key)
✔ Organização de dados em modelo relacional
💻 Tecnologias utilizadas:
• SQL
• MySQL
• SGBD (Sistema de Gerenciamento de Banco de Dados)
Esse projeto faz parte da minha jornada de aprendizado na área de Tecnologia da Informação e desenvolvimento de sistemas, buscando evoluir cada vez mais em Banco de Dados e desenvolvimento backend.
